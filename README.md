# AI vs Real Image Classifier

This is an **AI vs Real Image Classifier** that distinguishes between **AI-generated images** and **Real images**.  

The dataset of AI images and Real images was **generated using web scraping**.  
The web scraping code can be found here: [Web Scraping Repository](https://github.com/Swarna1806/WebScraping)

## Features
- Classifies an uploaded image as either **AI Image** or **Real Image**.
- Provides **real-time predictions** with confidence scores through a simple UI.
- Dataset creation is automated using **Selenium** and **BeautifulSoup** for scraping images.
